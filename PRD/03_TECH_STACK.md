# 03. Tech Stack

## Frontend
- Next.js 15
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Framer Motion
- GSAP
- Lenis
- Lucide Icons

## Backend
- Laravel 12 REST API
- Filament admin panel

## Database
- PostgreSQL
- Supabase Postgres is acceptable for managed hosting

## Storage
- Supabase Storage or S3-compatible storage

## Authentication
- Laravel session or token-based auth for the admin dashboard
- Owner-only access for admin routes

## Deployment
- Frontend: Vercel
- Backend: VPS

## Tooling Rules
- Use reusable components
- Use strict TypeScript
- Keep code modular and maintainable
- Optimize for performance and SEO from the start
