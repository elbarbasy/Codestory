# 01. Project Overview

## Product Name
CodeStory

## Product Type
Premium bilingual agency website with a private owner-only CMS dashboard.

## Vision
Build a website that feels like a modern SaaS product and converts visitors into qualified leads for web development services.

## Core Goal
The website must:
- create a strong first impression within 5–10 seconds,
- explain services clearly,
- showcase portfolio credibly,
- present pricing transparently,
- make it easy to contact the owner,
- allow the owner to manage all content through a private dashboard.

## Target Users
- Small businesses and UMKM
- Startup founders
- Personal brands
- Local companies needing company profile websites
- Prospects looking for landing pages, business websites, or custom web apps

## Business Objectives
- Increase inbound leads
- Improve trust and brand perception
- Show technical capability through premium motion design
- Support Indonesian and English audiences

## Scope for Initial Release
- Home
- Services
- Portfolio
- Pricing
- About
- Contact
- Privacy Policy
- Terms and Conditions
- 404 page
- Private admin dashboard

## Out of Scope for Initial Release
- Blog
- Public user registration
- Public comments
- Multi-tenant customer dashboard

## Key Product Principles
- Premium but fast
- Clear but elegant
- Conversion-focused
- Mobile-first responsive behavior
- Owner-managed content via CMS
- Strong visual identity based on CodeStory branding
