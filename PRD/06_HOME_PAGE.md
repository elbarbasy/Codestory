# 06. Home Page Specification

## Goal
The Home page must convince visitors within the first 5–10 seconds that CodeStory is a premium web development agency. Every section should build trust and encourage users to contact the business.

## Section Order
1. Terminal Loader
2. Transparent Navbar
3. Hero
4. Trusted By / Client Logos
5. Statistics
6. Services Bento Grid
7. Workflow
8. Featured Portfolio
9. Pricing Preview
10. Testimonials
11. FAQ
12. Final CTA
13. Footer

## Navbar
Desktop:
- Logo
- Home
- Services
- Portfolio
- Pricing
- About
- Contact
- Language Toggle (ID | EN)
- Primary CTA: Konsultasi Gratis

Mobile:
- Logo
- Language Toggle
- Hamburger Menu

## Hero
Headline:
"Website yang Tidak Hanya Indah, Tapi Juga Menghasilkan Klien."

Subheadline:
"Kami membantu bisnis, UMKM, dan startup membangun website modern yang cepat, profesional, dan berorientasi pada hasil."

Primary CTA:
- Konsultasi Gratis

Secondary CTA:
- Lihat Portfolio

Right Side:
- Floating browser mockup
- Technology badges (Next.js, Laravel, Supabase)
- Soft blue glow background

## Trusted By
Horizontal logo marquee for client logos and partners.

## Statistics
Animated counters:
- 50+ Project
- 98% Client Satisfaction
- 24/7 Support
- 3+ Years Experience

## Services
Display six premium cards:
- Company Profile
- Landing Page
- E-Commerce
- Custom Web App
- AI Integration
- Maintenance

## Workflow
Five-step timeline:
Consultation -> UI/UX Design -> Development -> Testing -> Launch

## Portfolio Preview
Display featured projects with:
- Preview image
- Technology stack
- Live Demo button
- GitHub button (optional)

## Pricing Preview
Show three highlighted packages with a button linking to the full pricing page.

## Testimonials
Infinite scrolling testimonials with customer names, roles, and ratings.

## FAQ
Accordion containing common questions about pricing, timeline, revisions, hosting, and maintenance.

## Final CTA
Headline:
"Siap Membangun Website Impian Anda?"

Buttons:
- Konsultasi Gratis
- Hubungi via WhatsApp

## Footer
Include:
- Logo
- Navigation
- Contact information
- Social media
- Copyright

## Acceptance Criteria
The Home page must load quickly, feel premium, support both desktop and mobile layouts, and guide users naturally toward contacting CodeStory.
