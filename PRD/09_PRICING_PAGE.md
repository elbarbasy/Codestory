# 09. Pricing Page

## Objective
Present transparent pricing, help visitors choose the right package, and maximize consultation requests through a clear comparison and interactive calculator.

## Page Structure
1. Hero
2. Package Tabs
3. Pricing Cards
4. Feature Comparison Table
5. Add-on Services
6. Live Price Calculator
7. FAQ
8. Final CTA

## Hero
Headline: "Pilih Paket yang Sesuai dengan Kebutuhan Anda"
Subtitle: Every package is scalable and can be customized.

## Package Categories
- Landing Page
- Company Profile
- Business
- Custom Web App

Switch categories without reloading the page.

## Pricing Cards
Each card must display:
- Package name
- Starting price
- Target customer
- Included features
- Estimated delivery time
- Revision policy
- CTA button

Highlight one package with a "Most Popular" badge.

## Feature Comparison
Compare:
- Responsive Design
- SEO Ready
- Admin Dashboard
- CMS
- Blog Support (future-ready)
- Database
- API Integration
- AI Integration
- Maintenance Period

## Add-on Services
Optional add-ons:
- Domain
- Hosting
- Business Email
- Payment Gateway
- Additional Pages
- Multi-language
- SEO Optimization
- Monthly Maintenance
- AI Chatbot

## Price Calculator
Allow visitors to:
- Select a package
- Enable add-ons
- View the total estimated price instantly

## FAQ
Address common pricing questions:
- Is hosting included?
- Can I request revisions?
- How long does development take?
- Can features be added later?

## Final CTA
Headline: "Masih Bingung Memilih Paket?"
Buttons:
- Konsultasi Gratis
- WhatsApp

## Acceptance Criteria
Pricing must be easy to compare, mobile friendly, transparent, and encourage visitors to contact CodeStory instead of leaving the page.
